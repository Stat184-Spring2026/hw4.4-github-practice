# HW #4.3 / HW #4.4 Repository

This repository contains my reproducible analysis project from HW #4.3 and demonstrates GitHub workflow skills for HW #4.4, including branching, issues, commits, pull requests, and project organization.

## Overview

This project uses Quarto and R to create a reproducible analysis document featuring data wrangling, simulation, and visualization. Topics include airport passenger traffic trends, Monte Carlo numerical integration, and calcium measurement analysis.

The repository also demonstrates practical use of GitHub version control tools.

## Interesting Insight

Passenger traffic across major global airports dropped sharply in 2020–2021 and then recovered strongly through 2025. Some airports, such as Dubai and Tokyo Haneda, showed especially rapid post-pandemic growth.
